Les API

Une Application Programming Interface est une Interface Applicative de Programmation qui permet d’établir des connexions entre plusieurs logiciels pour échanger des données.Plus techniquement, il s’agit d’un ensemble de fonctions qui vont permettre à un développeur d’utiliser simplement une application dans son programme.

Par exemple, vous avez besoin d'intégrer la gestion d'un moyen de paiment dans votre site, l'API de Paypal permet de gérer les services bancaires en l'intégrant directement dans votre site. Il en est de même pour des sites comme Youtube, SoundCloud, Viméo, Deezer, etc. C'est ce que nous allons vous présenter ici, avec Youtube, Deezer.
